# ARLiveBriefInterview

A native Android application utilizing Jetpack Compose and ARCore Geospatial API to host LLM-driven conversational interviews.

## Features
- **Clean Material 3 UI:** Easily paste in interview briefs (such as those generated by NotebookLM).
- **ARCore Geospatial Live View:** Real-world tracking ensuring the subject can freely look around while questions appear floating in physical space via AR overlays.
- **Automated Flow:** Uses native synthetic TTS to speak questions.
- **Media Recorder:** Records standard `.m4a` audio directly from the device's microphone, then accurately merges them sequentially into a single high-quality final transcript using Android's native `MediaMuxer` - No bloated third party libraries.

## Requirements
- Android SDK 34
- Minimum SDK 28
- A physical Android device supporting ARCore (emulators without special configuration will crash trying to open AR sessions).
- Google Cloud API Key with **ARCore API** enabled.

## Setup Instructions

1. Clone or download this project.
2. Open the project in **Android Studio**.
3. **Important:** Obtain an ARCore API Key:
   - Go to the [Google Cloud Console](https://console.cloud.google.com/).
   - Create a project (or select an existing one).
   - Enable the **ARCore API**.
   - Navigate to Credentials and Create an API Key.
4. Add your API Key to the project:
   - Open `app/src/main/AndroidManifest.xml`.
   - Locate the `com.google.android.ar.API_KEY` meta-data tag.
   - Replace `YOUR_GOOGLE_CLOUD_ARCORE_API_KEY` with your actual generated key.
5. Connect your physical Android phone (must support ARCore) and click Run in Android Studio.

## Usage
1. Open the app on your phone.
2. Paste an interview brief into the text box (e.g. from an LLM).
   *Questions should be numbered (e.g., `1. Query`) or separated by two newlines.*
3. Click "Start AR Live View Interview Session".
4. Ensure you grant Camera and Microphone (as well as Location) permissions.
5. Move your phone to initialize the Geospatial mesh. The app will sequence through your questions natively speaking them and recording your answers!
